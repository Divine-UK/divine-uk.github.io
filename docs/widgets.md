# Widgets

## Title

    {
      "id": "",
      "type": "title",
      "props": {
        "text": "Divine UK",
        "size": "h1",
        "align": "center",
        "color": "primary"
      }
    }
#### Properties

 - text
 - size - h1, h2, h3, h4
 - align - left, center, right
 - colour - primary, secondary, tertiary

## Image

    {
      "id": "",
      "type": "image",
      "props": {
        "src": "url",
        "aspect_ratio": "1.3",
        "link": "divineuk://",
        "alt_text": ""
      }
    }
#### Properties

 - src - Source url
 - aspect_ratio - 4/3; 16/10 etc.
 - link - Fir the future, when we want to link to another page in the app
 - alt_text - For accessibility
